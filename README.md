## 🧠 What is an Algorithm?

An **algorithm** is a step-by-step procedure to solve a problem efficiently.

It focuses on:

- ⏱️ Time Complexity → How fast the solution runs  
- 💾 Space Complexity → How much memory it uses  
- ✅ Correctness → Produces the right output  
- 🔄 Optimization → Making solutions faster & better  

---

## 🧊 Algorithm Concept Cube

| Dimension | Meaning |
|-----------|---------|
| TIME      | Execution speed (Big-O) |
| SPACE     | Memory usage |
| LOGIC     | Step-by-step correctness |
| OPTIMIZE  | Improving performance |

📦 **Algorithm = Logic + Time + Space**
